About

Edeson File Converter

Version 1.0.0

Developed by: ENDY EDESON

Edeson File Converter is a modern, fast, and secure client-side file conversion application. It allows you to convert images, documents, audio, video, and manipulate PDF files directly in your browser without uploading any files to a server. Built with privacy and performance in mind, it ensures your data stays on your device at all times.

Features

  • Image conversion, resizing, and transformation
  • Document format conversion (TXT, HTML, Markdown, JSON, CSV, XML)
  • Audio format conversion with trimming and volume control
  • Video format conversion with trimming and thumbnail capture
  • PDF tools: merge, split, rotate, and preview
  • 100% client-side processing - no server required
  • Dark and Light theme support
  • Responsive design for mobile and desktop
  • Conversion history tracking
  • Drag and drop file upload support
